# ThinkJS-RABC
ThinkJS-RABC 是一款基于[ThinkJS](https://thinkjs.org/)框架开发的简易权限管理系统,目前主要包含用户、用户组管理，菜单管理，字典、字典组管理，菜单分配，系统操作日志几大功能，页面布局采用Bootstrap布局，适应不同尺寸屏幕。
